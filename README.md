# Next.js Admin Dashboard

A fully responsive **admin dashboard** built with **Next.js**, **React**, and **Tailwind CSS**, featuring Firebase authentication, AI chat integration, and data visualization with Recharts. Designed as a starter scaffold for admin panels, analytics apps, and internal tools.

## **Features**
- Protected routes using **Firebase authentication**
- Responsive sidebar & topbar layout
- Dark/light **theme toggle** with localStorage persistence
- KPI cards, charts, tables, and lists
- AI chat integration (stubbed with `callAiChat`)
- Tailwind CSS for modern UI design
- Recharts for interactive charts
- Fully modular React component structure
- Ready for real backend API integration
